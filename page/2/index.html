socket.io封装了websocket，同时包含了其它的连接方式，比如Ajax。原因在于不是所有的浏览器都支持websocket，通过socket.io的封装，你不用关心里面用了什么连接方式。你在任何浏览器里都可以使用socket.io来建立异步的连接。

与服务端建立连接

这里要注意，因为是一个聊天系统，所以与服务端建立连接的方式不同于往常，这里用到的协议是HTTP WebSocket，从而实现持久连接。
简单的解释一下，这里的“持久”，是相对于HTTP这种“非持久”的协议来说的（阁下的意思是，HTTP的夫人会很羡慕WebSocket的夫人咯）。
通过阅读Ovear在知乎上的回答，大致说一下这两个协议之间的区别。

HTTP

HTTP的生命周期大概是这样的，一个request，一个response，这次请求就结束了；HTTP 1.1中进行了改进，增加了一个keep-alive，效果是在这次HTTP连接中，可以发送多个request，接受多个response，但本质上，request = response，也就是说，请求和响应永远是一一对应的，没有request时，服务端不能主动response。

WebSocket

当客户端与服务端完成协议升级以后（HTTP -> WebSocket），就建立了一个持久连接，有多持久呢？这个连接可以持续存在知道客户端或服务端某一方主动的关闭连接。与HTTP最大的不同是，此时的服务端可以主动推送消息给客户端咯。在这个项目中，我们用socket.io这个包模块来实现WebSocket，socket.io不仅实现了对WebSocket的封装，还将连同Ajax轮询和其他实时通信方式封装成了通用的接口，这么做的原因是，当服务器不支持WebSocket时，可以转换为其他的实现方式，啧啧啧，堪称纵享丝滑:kiss:

接受消息

接受消息这个逻辑有三种情况

  1. 自己发送的消息
  2. 别人发送的消息
  3. 系统的提示信息

莫慌，一个一个来看。

自己发送的消息

自己发送的消息直接显示在聊天消息的面板，接收自己发送的消息不用和后台交互，只需要告诉后台我给大家发了这条消息即可。当然啦，你也可以仿照微信对自己发送的消息进行处理：发送的瞬间将自己的消息添加聊天面板 -> 给旁边放个小菊花或者loading的字样 -> 与后台进行交互 -> 成功则隐藏小菊花；失败则将小菊花变成红色感叹号暗示用户发送失败。

没差 | 坚持走完这一场

发表于 2016-11-13 | 分类于 jotting

从小被长辈们告知，“这个社会是不公平的”，总是以为自己做好了足够的心理准备，可是到了节骨眼才发现，这些扭曲的事实足以颠覆我们的价值观。

「一」

“真没想到，同一所学校里面的人差距会这么大。”

自从上大学以来，听到最多，最有真切感受的大概就是这么一句话了。最先告诉我这个事实的人是我的一个老同学，一个很努力的女生，现在回想起来，脑海里对她的印象一直是她留在毕业照上的那张笑的很可爱的脸。高考之后她去了另一座城市，一所985院校，热门的专业，一个绝大多数人都梦寐以求的结果。一次偶然的机会和她取得了联系，才得知，她并没有我想像中过的那么好。

“当初同一届那么多人，走的最好的估计就是你了。”

“别这么说，如果真让我再选一次，我宁愿选一所普通的学校。”

“啊……?”

“来了这里才发现，自己真的是差太多了。”

“不要有压力嘛，本来就是人外有人啊，但是你本身也不差啊，努力一下就和那些人一样优秀了。”

“不不不，有些东西真的是没办法弥补的。我们班有一个妹子，学习特别认真，老是考年级前5，能力还特别强，每次做策划什么的，一个人几乎能搞定所有事……”

“学霸呗，哪个学校都有啊”，我打断了她，“你俩路子不同，再不济你也可以靠脸吃饭啊。”

“得了吧，她是我们学院的院花，追她的男生可以从学校的东门排到西门，最可气的是，他男朋友还特别好，俩人大一就好上了，人帅品好，还特别暖，虽然成绩没她那么好，但以后肯定是个好老公。”

“那这种女生不得成你们学校的公敌了”

“不是啊，妹子人特别好，我都特别喜欢和她待一起，很温柔知性的一个人。”

“她爸妈生了这么一个女儿，以后肯定发家致富啊。”

“她家北京的…三环以内…还有两套房…喏，这是她朋友圈。”

我接过手机，翻着看了看，没有嗅到一丝富家女的公主气息，更颠覆的是，她朋友圈分享的一些东西都是很有意义的，下乡支教，打工换宿……

你看，人和人的差距本来就是这么大。现实就是这样，往往在你非常努力的时候，给你泼这么一盆冷水。

「二」

“你相信付出多少就一定能收获到多少吗？”

还记得高考前的一次班会上，老师说，“这是你们人生中最后一次还称得上是公平的战役了，还不好好把握住机会。”

这个社会有这么不公平吗？

有，当然有。虽然还没有彻底进入社会，但是被比做“小社会”的大学生活，也着实给我上了一课。

远的不说，同一个学校内就会有这样的事情发生。

成绩和能力都很优秀的人，本可以保送到国内顶尖的高校读研，结果因为学校政策的原因，只保了本校；大家眼里公认的学霸，运气太差，挂掉了一门无关痛痒的课程，失去了保研资格；另一个软件专业的同学，本科期间自己写过的代码可能都没有100行，上课时间都以玩手机睡觉度过，靠着班干部学生会的“工作”和为老师的朋友圈点赞，一个对学术没有任何天赋和热情的人，不仅拿了三年的奖学金，保了研，还发朋友圈宣告世界自己有多牛逼。

我们都在往上爬，这旅程，越往后走，就越难，如履薄冰。可就当我们逆着凛冽的寒风，咬牙坚持的时候，看到有人吃着火锅唱着歌，开着传送就到达了看似比我们更高的地方。

“我能说脏话吗？”

“不能。”

「三」

“你刚才问我什么？”

“你相信付出多少就一定能收获到多少吗？”

“信。”

你看，那些一直在努力的同学，不是已经有了一个配得上他们的结果，就是正在为了一个更好的目标努力着。

我时常多么庆幸，身边能有这么一群靠着自己的努力不断前进的人，一直在提醒着我，努力的意义还是值得去相信并践行的；当然，也谢谢那些懒惰贪婪坚持投机取巧的人，不断的警示着我堕落的样子有多么的令人厌恶。

我不信那些钻了制度漏洞的人可以靠着“潜规则”过一辈子。

我相信那些专注于不断提升自己的人，最后都能到达想去的远方。

要走的路还很长，现在这些，都没差。

“若有天我不复勇往，能否坚持走完这一场”

至少在这一刻，“能”。
